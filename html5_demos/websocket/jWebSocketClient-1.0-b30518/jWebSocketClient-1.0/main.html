jWebSocket JavaScript Client

Demos and Examples

This web site demostrates the capabilities of the jWebSocket framework. You can operate this site either in an arbitrary folder of your web server (e.g. Apache) and access a stand-alone version of the jWebSocket Server jWebSocketServer.jar or you can run it within the web application jWebSocketAppServer.war package by simply deploying the .war-file in a servlet container like Tomcat or Jetty. For browsers which do not yet natively support WebSockets and the provided fallbacks please refer to the required server side provisions described at http://jwebsocket.org.

To integrate the jWebSocket client into your web application please embed the jWebSocket_Bundle.js file into your environment. In the demos provided with jWebSocket you will find numerous examples of how to use the jWebSocket API.
